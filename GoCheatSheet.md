## Методы пакета fmt

Вывод в консоль print

| Функция             | Описание                                                     |
| :------------------ | :----------------------------------------------------------- |
| fmt.Print(a, b)     | Выводит без переноса строки// fmt.Print("hi") → hi           |
| fmt.Println(a, b)   | Выводит с переносом строки, между аргументами пробел// fmt.Println("hi") → hi\n |
| fmt.Printf(f, a...) | Вывод по шаблону с форматированием// fmt.Printf("%s = %d", "x", 5) → x = 5 |

Формирование строки sprint

| Функция              | Описание                                                     |
| :------------------- | :----------------------------------------------------------- |
| fmt.Sprintf(f, a...) | Возвращает строку по шаблону (не выводит)// s := fmt.Sprintf("x=%d", 5) → "x=5" |
| fmt.Sprint(a...)     | Возвращает строку без шаблона// s := fmt.Sprint("a", 1) → "a1" |
| fmt.Sprintln(a...)   | Возвращает строку с переносом в конце// s := fmt.Sprintln("hi") → "hi\n" |

Чтение ввода scan

| Функция              | Описание                                                     |
| :------------------- | :----------------------------------------------------------- |
| fmt.Scan(&a)         | Читает значения разделённые пробелом/переносом// fmt.Scan(&x, &y) |
| fmt.Scanln(&a)       | Читает до конца строки// fmt.Scanln(&x)                      |
| fmt.Scanf(f, &a)     | Читает по шаблону// fmt.Scanf("%d %d", &a, &b)               |
| fmt.Sscanf(s, f, &a) | Разбирает строку по шаблону// fmt.Sscanf("5 10", "%d %d", &a, &b) |

Ошибки error

| Функция                 | Описание                                                     |
| :---------------------- | :----------------------------------------------------------- |
| fmt.Errorf(f, a...)     | Создаёт error с форматированным текстом// err := fmt.Errorf("not found: %d", id) |
| fmt.Fprintf(w, f, a...) | Пишет в io.Writer (файл, буфер, stderr)// fmt.Fprintf(os.Stderr, "err: %s", e) |

Глаголы форматирования (verbs)

| Verb | Тип               | Пример                           |
| :--- | :---------------- | :------------------------------- |
| %v   | любой тип, дефолт | fmt.Printf("%v", 42) → 42        |
| %T   | тип переменной    | fmt.Printf("%T", 42) → int       |
| %d   | целое число       | fmt.Printf("%d", 42) → 42        |
| %f   | float             | fmt.Printf("%.2f", 3.14) → 3.14  |
| %s   | строка            | fmt.Printf("%s", "hi") → hi      |
| %q   | строка в кавычках | fmt.Printf("%q", "hi") → "hi"    |
| %b   | двоичное число    | fmt.Printf("%b", 5) → 101        |
| %x   | шестнадцатеричное | fmt.Printf("%x", 255) → ff       |
| %p   | указатель         | fmt.Printf("%p", &x) → 0xc000... |
| %t   | bool              | fmt.Printf("%t", true) → true    |

**Спецификаторы `fmt.Printf`**

| Verb | Тип                 | Пример                            |
| ---- | ------------------- | --------------------------------- |
| `%v` | любой тип           | `Printf("%v", 42)` → `42`         |
| `%T` | тип переменной      | `Printf("%T", 42)` → `int`        |
| `%t` | bool                | `Printf("%t", true)` → `true`     |
| `%d` | целое, десятичная   | `Printf("%d", 42)` → `42`         |
| `%b` | целое, двоичная     | `Printf("%b", 5)` → `101`         |
| `%o` | целое, восьмеричная | `Printf("%o", 8)` → `10`          |
| `%x` | целое, hex a-f      | `Printf("%x", 255)` → `ff`        |
| `%X` | целое, hex A-F      | `Printf("%X", 255)` → `FF`        |
| `%c` | символ по коду      | `Printf("%c", 65)` → `A`          |
| `%q` | символ в кавычках   | `Printf("%q", 'Ы')` → `'Ы'`       |
| `%f` | float обычный       | `Printf("%f", 3.14)` → `3.140000` |
| `%e` | float экспонента    | `Printf("%e", 3.14)` → `3.14e+00` |
| `%g` | float авто          | `Printf("%g", 3.14)` → `3.14`     |
| `%s` | строка              | `Printf("%s", "hi")` → `hi`       |
| `%p` | указатель           | `Printf("%p", &x)` → `0xc000...`  |

------

**Ширина и точность**

| Формат   | Описание              | Результат   |
| -------- | --------------------- | ----------- |
| `%.2f`   | 2 знака после запятой | `3.14`      |
| `%9.2f`  | ширина 9, точность 2  | `·····3.14` |
| `%-9.2f` | выравнивание влево    | `3.14·····` |
| `%09d`   | заполнить нулями      | `000000042` |

------

**Главное:** `Sprintf` работает так же но возвращает строку:

```go
s := fmt.Sprintf("%.2f", 3.14) // s = "3.14"
```

## Методы пакета strconv

Строка → число parse

| Функция                         | Описание                                                     |
| :------------------------------ | :----------------------------------------------------------- |
| strconv.Atoi(s)                 | Строка → int. Самая частая функцияn, err := strconv.Atoi("42") // n=42strconv.Atoi("abc") // err != nil |
| strconv.ParseInt(s, base, bits) | Строка → int64, можно указать основание системы счисленияn, err := strconv.ParseInt("FF", 16, 64) // n=255n, err := strconv.ParseInt("42", 10, 64) |
| strconv.ParseFloat(s, bits)     | Строка → float64f, err := strconv.ParseFloat("3.14", 64)     |
| strconv.ParseBool(s)            | Строка → bool. Принимает: "1", "t", "true", "0", "f", "false"b, err := strconv.ParseBool("true") // b=true |

Число → строка format

| Функция                                 | Описание                                                     |
| :-------------------------------------- | :----------------------------------------------------------- |
| strconv.Itoa(n)                         | int → строка. Обратная Atois := strconv.Itoa(42) // s="42"   |
| strconv.FormatInt(n, base)              | int64 → строка, можно указать основаниеs := strconv.FormatInt(255, 16) // s="ff"s := strconv.FormatInt(5, 2) // s="101" |
| strconv.FormatFloat(f, fmt, prec, bits) | float64 → строка. fmt: 'f' обычный, 'e' экспонента, 'g' автоs := strconv.FormatFloat(3.14, 'f', 2, 64) // "3.14" |
| strconv.FormatBool(b)                   | bool → строкаs := strconv.FormatBool(true) // s="true"       |

Прочее other

| Функция            | Описание                                                     |
| :----------------- | :----------------------------------------------------------- |
| strconv.Quote(s)   | Оборачивает строку в Go-кавычки, экранирует спецсимволыstrconv.Quote("hi\n") // `"hi\n"` |
| strconv.Unquote(s) | Убирает кавычки, обратное Quotestrconv.Unquote(`"hello"`) // "hello", nil |

Все Parse-функции возвращают два значения: результат и error. Всегда проверяй err != nil — если ст

строка не конвертируется, программа не упадёт сама, но получишь неверное значение.

## Массивы и слайсы

Массив (array) array

| Синтаксис              | Описание                                           |
| :--------------------- | :------------------------------------------------- |
| var a [5]int           | Массив из 5 int, заполнен нулями// [0 0 0 0 0]     |
| a := [3]int{1, 2, 3}   | Массив с начальными значениями// [1 2 3]           |
| a := [...]int{1, 2, 3} | Размер определяется автоматически// [1 2 3], len=3 |
| a[0]                   | Доступ по индексу (с 0)a[0] = 10                   |
| len(a)                 | Длина массиваlen([3]int{1,2,3}) // 3               |

Массив — фиксированного размера. Размер является частью типа: [3]int и [5]int — разные типы.

Срез (slice) — объявление slice declare

| Синтаксис               | Описание                                              |
| :---------------------- | :---------------------------------------------------- |
| var s []int             | Пустой срез, nil// s == nil, len=0                    |
| s := []int{1, 2, 3}     | Срез с начальными значениями// [1 2 3]                |
| s := make([]int, 5)     | Срез длиной 5, заполнен нулями// [0 0 0 0 0]          |
| s := make([]int, 3, 10) | Длина 3, ёмкость 10// len=3, cap=10                   |
| s := a[1:4]             | Срез из массива/среза, индексы [1,4)// элементы 1,2,3 |
| s := a[:3]              | От начала до индекса 3// элементы 0,1,2               |
| s := a[2:]              | От индекса 2 до конца// элементы 2,3,4...             |

Операции со срезом slice ops

| Синтаксис       | Описание                                                     |
| :-------------- | :----------------------------------------------------------- |
| len(s)          | Количество элементовlen([]int{1,2,3}) // 3                   |
| cap(s)          | Ёмкость (сколько влезет без реаллокации)cap(make([]int,3,10)) // 10 |
| append(s, x)    | Добавить элемент в конецs = append(s, 4)                     |
| append(s, a...) | Добавить все элементы другого срезаs = append(s, []int{5,6,7}...) |
| copy(dst, src)  | Копирует элементы, возвращает кол-во скопированныхn := copy(dst, src) |
| s[i:j]          | Подсрез [i, j) — ссылается на тот же массивs[1:3] // элементы 1 и 2 |

Обход среза range

| Синтаксис                   | Описание                                      |
| :-------------------------- | :-------------------------------------------- |
| for i := 0; i < len(s); i++ | Классический цикл по индексуfmt.Println(s[i]) |
| for i, v := range s         | Индекс и значениеfmt.Println(i, v)            |
| for _, v := range s         | Только значениеfmt.Println(v)                 |
| for i := range s            | Только индексfmt.Println(i)                   |

Массив vs Срез

| Свойство           | Массив                  | Срез                  |
| :----------------- | :---------------------- | :-------------------- |
| Размер             | фиксированный           | динамический          |
| Передача в функцию | копируется целиком      | передаётся ссылка     |
| Нулевое значение   | [0 0 0 ...]             | nil                   |
| Когда использовать | размер известен заранее | в большинстве случаев |

Срез — это не копия данных, а ссылка на массив под капотом (указатель + длина + ёмкость). Изменение элементов среза меняет исходный массив.
