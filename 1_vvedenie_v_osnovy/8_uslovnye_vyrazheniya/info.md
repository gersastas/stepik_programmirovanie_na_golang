Условные выражения в Go включают операции отношения и логические операции. Они позволяют создавать условия, которые возвращают логическое значение типа `bool`: `true` (если условие истинно) или `false` (если условие ложно).

### Операции отношения

Операции отношения позволяют сравнивать два значения. В языке Go доступны следующие операции отношения:

- **==**: Операция "равно". Возвращает `true`, если оба операнда равны, и `false`, если они не равны:

  ```go
  package main
  import "fmt"
  
  func main() {
      var a int = 8
      var b int = 3
      var c bool = a == b
      fmt.Println(c) // false
  }
                    
  ```

- **>**: Операция "больше чем". Возвращает `true`, если первый операнд больше второго, и `false`, если первый операнд меньше или равен второму:

  ```go
  var a int = 8
  var b int = 3
  var c bool = a > b  // true
                    
  ```

- **<**: Операция "меньше чем". Возвращает `true`, если первый операнд меньше второго, и `false`, если первый операнд больше или равен второму:

  ```go
  var a int = 8
  var b int = 3
  var c bool = a < b  // false
                    
  ```

- **<=**: Операция "меньше или равно". Возвращает `true`, если первый операнд меньше или равен второму, и `false`, если первый операнд больше второго:

  ```go
  var a int = 8
  var b int = 3
  var c bool = a <= b // false
                    
  ```

- **>=**: Операция "больше или равно". Возвращает `true`, если первый операнд больше или равен второму, и `false`, если первый операнд меньше второго:

  ```go
  var a int = 8
  var b int = 3
  var c bool = a >= b // true
                    
  ```

- **!=**: Операция "не равно". Возвращает `true`, если первый операнд не равен второму, и `false`, если они равны:

  ```go
  var a int = 8
  var b int = 3
  var c bool = a != b // true
  var d bool = a != 8 // false
                    
  ```

Как правило, операции отношения используются в условных конструкциях, таких как `if...else`, для принятия решений на основе условий. Рассмотрим следующий пример:

```go
package main
import "fmt"

func main() {
    var a, b int = 8, 3

    if a > b {
        fmt.Println("a больше b")
    } else {
        fmt.Println("a не больше b")
    }
}
                  
```

В этом примере условие `a > b` проверяется с помощью операции отношения, и в зависимости от результата выводится соответствующее сообщение.

### Логические операции

Логические операции используются для сравнения двух условий. Они обычно применяются к операциям отношения и объединяют несколько из них. В Go есть три основные логические операции:

- **!** (операция отрицания, логическое НЕ):

  Инвертирует значение. Если операнд равен `true`, то возвращает `false`, иначе возвращает `true`.

  ```go
  var a bool = true
  var b bool = !a    // false
  var c bool = !b    // true
                    
  ```

- **&&** (конъюнкция, логическое И):

  Возвращает `true`, если оба операнда истинны (не равны `false`). Возвращает `false`, если хотя бы один операнд ложен (равен `false`).

  ```go
  var b bool = 4 > 5 && 6 > 8    // false
  var c bool = 3 <= 5 && 10 > 8  // true
                    
  ```

- **||** (дизъюнкция, логическое ИЛИ):

  Возвращает `true`, если хотя бы один операнд истинный (не равен `false`). Возвращает `false`, если оба операнда ложны (равны `false`).

  ```go
  var b bool = 4 > 5 || 6 > 8    // false
  var c bool = 3 == 5 || 10 > 8  // true
                    
  ```

Эти операции часто используются для сложных условий в конструкции `if` или при объединении нескольких условий для более гибкой логики.
