### **Циклы в Go**

В Go существует только одна конструкция для работы с циклами — это оператор `for`.

Базовый синтаксис этого цикла выглядит следующим образом:

```less
for [инициализация счетчика]; [условие]; [изменение счетчика]{
    // действия
}

                  
```

Пример простого цикла:

```go
package main

import "fmt"

func main() {
	sum := 0
	// Цикл от 1 до 9
	for i := 1; i < 10; i++ {
		sum += i
	}
	fmt.Println(sum)
}

                  
```

Цикл `for` состоит из трех частей:

- **Инициализация счетчика**: `i := 1` — здесь мы объявляем переменную, которая будет использоваться в цикле.
- **Условие**: `i < 10` — цикл продолжается, пока это условие возвращает `true`.
- **Изменение счетчика**: `i++` — переменная счетчика увеличивается на единицу после каждого шага цикла.

 

Важно отметить, что мы не обязаны использовать все три части при объявлении цикла. Например, можно вынести инициализацию переменной за пределы цикла:

```go
var i = 1
for ; i < 10; i++ {
    fmt.Println(i * i)
}

                  
```

Также можно оставить условие в цикле, а увеличение счетчика переместить в тело цикла:

```go
var i = 1
for ; i < 10; {
    fmt.Println(i * i)
    i++ // увеличение счетчика внутри цикла
}

                  
```

Если в цикле используется только условие, то его можно записать еще проще:

```go
var i = 1
for i < 10 {
    fmt.Println(i * i)
    i++ // увеличение счетчика
}

                  
```

Такой цикл является аналогом цикла `while` в других языках программирования.

Также можно создать бесконечный цикл, который будет работать до тех пор, пока не произойдёт внешнее прерывание:

```go
for {
    // действия
}

                  
```

**Интересное замечание:** иногда можно использовать такую конструкцию для ввода данных, но будьте внимательны — в этом примере ошибки ввода не обрабатываются:

```go
var n int
// считываем числа до тех пор, пока не введен 0
for fmt.Scan(&n); n != 0; fmt.Scan(&n) {
	fmt.Println(n)
}

                  
```

### **Операторы break и continue**

В процессе выполнения цикла может возникнуть ситуация, когда необходимо досрочно завершить текущую итерацию или выйти из всего цикла. Для этих целей в Go используются операторы `continue` и `break`.

Оператор `continue` позволяет пропустить текущую итерацию цикла и перейти к следующей. Это полезно, когда необходимо выполнить только часть кода цикла при соблюдении определенного условия. Например, давайте посчитаем сумму нечетных чисел в диапазоне от 1 до 10. Если находим четное число, мы пропускаем его и переходим к следующей итерации:

```go
var sum = 0

for i := 1; i <= 10; i++ {
    if i % 2 == 0 {
        continue // пропускаем четные числа
    }
    sum += i
}
fmt.Println("Сумма: ", sum) // Сумма: 25

                  
```

В этом примере, если число четное (например, 2, 4, 6 и т.д.), оператор `continue` пропускает выполнение оставшейся части тела цикла и переходит к следующей итерации.

Оператор `break` используется для полного завершения выполнения цикла. Он немедленно прерывает выполнение цикла и выходит из него, не дождавшись окончания всех итераций. Рассмотрим пример, где мы хотим суммировать числа от 1 до 9, но выйти из цикла, как только число станет больше 4:

```go
var sum = 0

for i := 1; i <= 9; i++ {
    if i > 4 {
        break // выходим из цикла, если число больше 4
    }
    sum += i
}
fmt.Println("Сумма: ", sum) // Сумма: 10

                  
```

В этом примере цикл завершится, как только переменная `i` станет больше 4, благодаря оператору `break`. В результате, в сумме окажутся только числа 1, 2, 3 и 4.
